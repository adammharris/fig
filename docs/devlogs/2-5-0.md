# `fig` version 2.5.0 "Parametric Embed API"

This release expands embedded config extraction with comprehensive archetype support and a refined parametric API.

## Embedded Config: HTML & Fenced Variants

When `fig` embeds config inside a host document (markdown, HTML), it now recognizes — and the CLI can explicitly target — a richer set of delimiter conventions:

**HTML Data Islands** — `<script type="application/<lang>">` (§ [Extract Config from Web Pages](../docs.md#extract))
- Typed script blocks for JSON-LD, YAML, TOML, or Fig config buried in `<head>` or anywhere mid-document
- Used widely in web publishing to carry machine-readable metadata without rendering visible markup
- All four supported formats: Fig, YAML, JSON, TOML

**HTML Code Blocks** — `<pre><code class="language-<lang>">`
- Visible, highlighted code blocks that *also* serve as the source of truth
- Spans `<code>` alone or `<pre><code>` pairs; class list recognizes `language-<lang>` and `lang-<lang>` tokens
- Content is HTML-entity-encoded (`&lt;`, `&gt;`, `&amp;`); edits preserve original byte-for-byte encoding on untouched bytes
- All four formats supported

**Fenced Code Blocks** — ` ```<lang> ` (now explicitly typed)
- Markdown ` ``` ` fence with language tag — already supported, now exposed as explicit archetypes
- Complements the original bare-`---` frontmatter family
- YAML, JSON, TOML, Fig

## ABI Additions

The C ABI `FigEmbedType` enum grows 16 new values, covering:
- Fenced + frontmatter variants in JSON, TOML, and Fig (markdown already had YAML via bare `---`)
- HTML script variants for all four formats
- HTML code variants for all four formats
- Plus TOML preset (`+++` delimiter)

All additions are backwards compatible; the function surface is unchanged.

## CLI

Use `--embed` with any of:
- `html-script`, `html-script-yaml`, `html-script-json`, `html-script-toml`
- `html-code`, `html-code-yaml`, `html-code-json`, `html-code-toml`
- `fenced-yaml`, `fenced-json`, `fenced-toml`, `fenced-fig`
- `md-json`, `md-toml`, `md-fig` (markdown frontmatter)
- `frontmatter-json` (aliases `;;;`), `frontmatter-toml` (aliases `+++`)
- `endmatter`, `endmatter-yaml`

## Example

Extract YAML config from an HTML script island:

```bash
fig get config.html --embed html-script-yaml
```

Edit the JSON embedded in visible highlighted code:

```bash
fig set config.html some.key value --embed html-code-json
```
